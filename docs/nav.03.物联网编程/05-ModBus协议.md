# 05-Modbus 协议介绍

MODBUS 是 OSI 模型第 7 层上的应用层报文传输协议，它在连接至不同类型总线或网络的设备之间提供客户机/服务器通信。
它主要用于工业自动化设备通信。
MODBUS可以在基于串行链路和以太 TCP/IP 网络的 MODBUS 上可以进行通信，也就是说，可以使用串口线或者网线链接两端设备，双方约定使用modbus协议去通信。
