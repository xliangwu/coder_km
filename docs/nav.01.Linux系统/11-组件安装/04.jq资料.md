# 04.jq手册

## Linux安装
>安装EPEL源
```bash
yum -y install epel-release
yum -y install jq
```

## 文档

## 示例

> 过滤数据的长度，然后求p50、p95
head data_1_2.txt |jq -cr '{recordId:.recordId,cnlu:.cnlu,size:.cnlu|length} |select(.size>19) | .recordId,.size,.cnlu[.size*0.5|floor].duration,.cnlu[.size*0.95|floor].duration' |sed '{N;N;N;s/\n/,/g}'
