# 10.Streamlit组件介绍

Streamlit是一个用于快速构建和部署数据应用的Python库。它的设计理念是使数据科学家能够快速而简便地创建交互式数据可视化和Web应用程序，而无需大量的前端开发经验。

## 1.安装
```bash
pip install streamlit
```

### 2.快速上手
```python
import streamlit as st

# 设置页面标题
st.title('简单的输出应用程序')

# 输出一行文字
st.write("欢迎使用Streamlit！")

```
